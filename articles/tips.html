Fase Inicial

  1. READY – Preparar suas cartas da mesa.
  2. SET – Efeitos do início do turno acontecem.
  3. DRAW – Compre uma carta. (Pule essa etapa no primeiro turno do jogo.)

Fase Principal

Escolha o quanto quiser (exceto que esteja escrito o contrário), em qualquer ordem:
  • Uma vez por turno, adicionar uma carta virada para baixo ao seu tinteiro.
  • Jogar uma carta.
  • Usar a habilidade de um personagem que não precisa ser virado.
  • Usar a habilidade de um item.
  • Usar a habilidade de uma localização.
  • Mover um personagem para uma localização.
  • COm um personagem que estava em jogo no início do seu turno:
    • Quest -ou-
    • Desafiar uma location location ou um personagem exerted -ou-
    • Usar uma habilidade que requeira que o personagem vire.

Fase Final

Efeitos no fim do turno acontecem e caso não tenha cartas no baralho você perde.

Ferramentas úteis

Digitais

  • dreamborn.ink - Dreamborn é o principal site que os jogadores usam hoje para procurar cartas e montar decks
  • inkdecks.com - InkDecks é o site com listas de decks que jogam torneio, com estatísticas
  • ligalorcana.com.br - Marketplace com diversas lojas. É possível encontrar cartas e comprá-las através de um sistema seguro
  • tcgplayer.com - Marketplace internacional para comprar cartas. Utilizado pelos jogadores para ver preços e compará-los
  • Disney Lorcana TCG Companion (Play Store) - Aplicativo oficial do Lorcana para pesquisar cartas, gerenciar sua coleção, criar decks e marcar a quantidade de lore durante um jogo
  • lorcast.com - Não muito utilizado, mas é um ótimo site para pequisar cartas, com um sistema avançado de pesquisa
  • Metafy - Site para ter dicas de jogadores profissionais, quase nenhum conteúdo em português
  • duels.ink - Site não oficial para jogar. Jogos rápidos e ranqueados.
  • Lorcanito - Site não oficial para jogar. Jogos rápidos e ranqueados, com possibilidade de jogar contra bots.

Itens úteis

  • Dados (de 6 lados é o mais comum) - pelo menos 4 desses dados para marcar o dano nos personagens e locations
  • Shields - proteção de plástico para colocar as cartas dentro e protegê-las
  • Playmat - "tapete", como um mousepad para jogar em cima e impedir que a mesa suje a carta
  • Deckbox - caixa para guardar o deck
